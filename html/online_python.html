Starte jetzt direkt mit der Programmierumgebung online-python

Wir fangen jetzt direkt an mit dem Programmieren und das ohne Installationen oder Einrichtung auf euren Computern.

Die Webseite online-python.com bietet uns die Möglichkeit Python Code zu schreiben und direkt auszuführen. Es gibt dabei 3 Felder, die wir verwenden können.

In das obere Feld, welches unter dem Reiter "main.py" beginnt wird das Skript (Programm) geschrieben. Darunter befindet sich ein grünes Feld mit der Aufschrift "Run", mit welchem das Progamm gestartet wird. In dem unteren Feld, einem Terminal oder auch als Kommando-Zeile bekannt, kann der Computer mit uns kommunizieren. Hier werden Ausgaben getätigt und auch Eingaben von uns an den Computer getätigt.
